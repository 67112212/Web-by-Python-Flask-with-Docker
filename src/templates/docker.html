หน้า Docker

สรุปความรู้เกี่ยวกับการสร้าง Docker Image:

  • สร้าง Dockerfile เพื่อกำหนด base image และคำสั่งติดตั้ง
  • ใช้คำสั่ง docker build -t yourname/webdemo .
  • รัน container ด้วย docker run -p 80:80 yourname/webdemo
  • push image ขึ้น Docker Hub ด้วย docker push yourname/webdemo
