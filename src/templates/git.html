หน้า Git

สรุปความรู้การใช้งาน GitHub ขั้นพื้นฐาน:

  • git init – สร้าง repository
  • git add . – เพิ่มไฟล์
  • git commit -m "message" – บันทึกการเปลี่ยนแปลง
  • git branch -M main – ตั้งชื่อ branch
  • git remote add origin ... – เชื่อม GitHub
  • git push -u origin main – อัพโหลดโค้ดขึ้น GitHub
